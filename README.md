Weather App
Description

A React app built with Vite that shows current weather data and clothing items based on temperature. The app uses reusable components, modals, and utility functions to manage weather data and item display.

Technologies:

React, Vite, JavaScript, CSS, Weather API, Normalize.css

Features:

- Displays weather in Fahrenheit
- Shows default clothing items
- Modal components for item details and forms
- Ability to add other clothing items
- Toggle Switch to change from Celsius to Fahrenheit
- Expands API-driven features such as comments or advanced search.
- Enhances UI/UX with light/dark mode themes.
- Optimizes performance with React.lazy and code splitting.
